# ClawBro

ClawBro is the top-level application package for the ClawBro runtime and CLI.

This package provides the public `clawbro` binary entrypoint and currently hosts:

- the CLI (`clawbro setup`, `clawbro serve`, `clawbro doctor`, `clawbro status`)
- the gateway runtime
- channel integrations
- team orchestration

The repository-level product documentation lives in the root [README](../../README.md).
